## Getting Started

### Prerequisites

프로젝트 실행방법

- yarn

  ```sh
  npm install --global yarn
  ```

  ```
  yarn start
  ```

### Installation

1. API [cockTailDB](https://www.thecocktaildb.com/api.php)
2. Clone the repo
   ```sh
   git clone https://github.com/jay08111/cockTailBar.git
   ```
3. Install yarn packages
   ```sh
   yarn install
   ```
